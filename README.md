# MediScan AI 🏥

AI-powered medical imaging diagnostic platform built for the AMD Developer Hackathon.

## What it does
- 🔬 Detects **skin cancer** (7 classes), **pneumonia**, and **eye diseases** from medical images
- 🤖 **LangChain AI agent** explains diagnoses in plain language
- 💬 Answers follow-up symptom questions
- 📍 Recommends nearby doctors via Google Maps

## Tech Stack
PyTorch • EfficientNet-B0 • LangChain • Gradio • FastAPI • HuggingFace Spaces

## Datasets
- Skin Cancer: 9,577 images, 7 classes
- Chest X-Ray: 5,216 images
- Eye Diseases: 4,217 images
